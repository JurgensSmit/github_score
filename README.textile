h1. Smithub

Smithub was created using the Github API to generate a score based on the events feed.

+1 For every watched repository
+10 Each created repository, branch, or tag.
+15 Any git push to a Repository.


h2. Ruby on Rails

This application requires:

* Ruby version 1.9.3
* Rails version 3.2.13


h2. Development

* Template Engine: ERB
* Testing Framework: RSpec and Factory Girl
* Front-end Framework: Zurb Foundation
